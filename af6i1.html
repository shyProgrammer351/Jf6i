문제

2km까지 기본료가 3800원이고 132m 당 100원이 부과된다고 가정할 때, 거리(m)를 입력하면 택시비를 계산하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
택시의 이동거리(m)를 입력하세요: 2000
2000m 이동시 택시요금은 3800원입니다


택시의 이동거리(m)를 입력하세요: 2001
2001m 이동시 택시요금은 3900원입니다


택시의 이동거리(m)를 입력하세요: 3600
3600m 이동시 택시요금은 5100원입니다

알고리즘

 프로그램 시작 
    이동거리를 입력받기 
    택시요금 계산 
    택시요금 출력 
 프로그램 종료 
 
 이동거리를 입력받는 함수 시작 
    사용자에게 이동거리를 입력받기 
 이동거리를 입력받는 함수 종료 
 
 택시요금 계산하는 함수 시작 
    이동거리가 2km 미만이면 2km로 초기화 
    택시요금은 2km까지 기본료가 3800원이고 132m 당 100원 부과 
 택시요금 계산하는 함수 종료

프로그램 코드 

	// 파일명 : ./Chapter06/TaxiFare.c 

	#include <stdio.h> 

	 

	// 이동거리를 입력받는 함수 시작 

1	int getDistance() 

	{ 

		int distance = 0; 

		// 사용자에게 이동거리를 입력받기 

		printf( "택시의 이동거리(m)를 입력하세요: " ); 

		// 2000을 입력한다고 가정 

		scanf( "%d", &distance ); 		 

	 

		// 이동거리를 입력받는 함수 종료 

2		return distance;							 

	} 

	 

	// 택시요금 계산하는 함수 시작 

3	int calculateTaxiFare( int distance ) 

	{ 

		int taxiFare = 0; 

		 

		// 이동거리가 2km 미만이면 2km로 초기화 

		distance = ( distance < 2000 ) ? 2000 : distance; 

	 

		// 택시요금은 2km까지 기본료가 3800원이고 132m 당 100원 부과 

		taxiFare = 3800 + ( distance - 2000 + 131 ) / 132 * 100; 

	 

		// 택시요금 계산하는 함수 종료 

4		return taxiFare;							 

	} 

	 

	// 프로그램 시작 

5	int main()									 

	{ 

		int distance = 0, taxiFare = 0; 

	 

		// 이동거리를 입력받기 

6		distance = 								 

7			   getDistance();		  

	 

		// 택시요금 계산 

8		taxiFare = 								 

9			   calculateTaxiFare( distance );	  

	 

		// 택시요금 출력 

10		printf( "%dm 이동시 택시요금은 %d원입니다 ", distance, taxiFare ); 

		 

		// 프로그램 종료 

11		return 0;								 

	}
