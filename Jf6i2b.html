문제

현재 기온을 입력하면 그 결과를 바탕으로 한파 경보, 한파 주의보, 폭염 주의보, 폭염 경보를 출력하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 참고로, 현재 기온이 
-15도 이하이면 한파 경보를 발효하고,
-13도 이하이면 한파 주의보를 발효하고,
35도 이상이면 폭염 경보를 발효하고,
33도 이상이면 폭염 주의보를 발효한다고 가정합니다.
섭씨 -100도부터 100도 사이에서 현재 기온은 몇 도인가요 : 18
오늘 날씨는 그나마 살만하네요 


섭씨 -100도부터 100도 사이에서 현재 기온은 몇 도인가요 : -14
일부 지방에 한파 주의보가 발효중입니다. 


섭씨 -100도부터 100도 사이에서 현재 기온은 몇 도인가요 : 35
일부 지방에 폭염 경보가 발효중입니다.

코드 빈칸 채우기

import java.util.Scanner; 
 
public class Weather 
{ 
	public static void main( String[] args ) { 
		Scanner scan = new Scanner( System.in ); 
		int temperature = 0; 
 
		System.out.print( "섭씨 -100도부터 100도 사이에서 현재 기온은 몇 도인가요 : " ); 
		temperature = scan.nextInt(); 
	 
		 (  temperature  -15 ) { 
			System.out.print( "일부 지방에 한파 경보가 발효중입니다. " ); 
		} 
		 (  ( -15  temperature )  ( temperature  -13 ) ) { 
			System.out.print( "일부 지방에 한파 주의보가 발효중입니다. " ); 
		} 
		 ( 35  temperature ) { 
			System.out.print( "일부 지방에 폭염 경보가 발효중입니다. " ); 
		} 
		 (  ( 33  temperature )  ( temperature  35 ) ) { 
			System.out.print( "일부 지방에 폭염 주의보가 발효중입니다. " ); 
		} 
		 { 
			System.out.print( "오늘 날씨는 그나마 살만하네요 " ); 
		} 
		 
	} 
}
