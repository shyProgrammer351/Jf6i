문제

사용자가 입력한 나이에 맞는 버스요금을 출력하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
참고로, 버스요금은 다음과 같습니다. 
* 영유아(만5세 이하)는 무료 
* 초등학생(만6세 - 만12세)은 현금기준 450원 
* 청소년(만13세 - 만18세)은 현금기준 1,000원 
* 성인(만19세 이상)은 현금기준 1,300원
나이를 입력하세요: 20
버스요금은 1,300원입니다. 


나이를 입력하세요: 18
버스요금은 1,000원입니다. 


나이를 입력하세요: 10
버스요금은 450원입니다.

코드 빈칸 채우기

import java.util.Scanner; 
 
public class BusFare 
{ 
	public static void main( String[] args ) { 
		Scanner scan = new Scanner( System.in ); 
		int age = 0; 
 
		System.out.print( "나이를 입력하세요: " ); 
		age = scan.nextInt(); 
 
		 ( age  5 ) { 
			System.out.print( "버스요금은 무료입니다." ); 
		} 
		 ( ( 6  age )  ( age  12 ) ) { 
			System.out.print( "버스요금은 450원입니다." ); 
		} 
		 ( ( 13  age )  ( age  18 ) ) { 
			System.out.print( "버스요금은 1,000원입니다." ); 
		} 
		 { 
			System.out.print( "버스요금은 1,300원입니다." ); 
		} 
 
	} 
}
