문제

2km까지 기본료가 3800원이고 132m 당 100원이 부과된다고 가정할 때, 거리(m)를 입력하면 택시비를 계산하는 문제입니다 이를 해결하는 다음 프로그램에 대해 빈칸을 채우세요 
택시의 이동거리(m)를 입력하세요: 2000
2000m 이동시 택시요금은 3800원입니다


택시의 이동거리(m)를 입력하세요: 2001
2001m 이동시 택시요금은 3900원입니다


택시의 이동거리(m)를 입력하세요: 3600
3600m 이동시 택시요금은 5100원입니다

코드 빈칸 채우기

#include <stdio.h> 
 
 getDistance() 
{ 
	int distance = 0; 
	printf( "택시의 이동거리(m)를 입력하세요: " ); 
	scanf( "%d", &distance ); 		 
 
	 distance;							 
} 
 
 calculateTaxiFare( int distance ) 
{ 
	int taxiFare = 0; 
	 
	distance = ( distance < 2000 ) ? 2000 : distance; 
 
	taxiFare = 3800 + ( distance - 2000 + 131 ) / 132 * 100; 
 
	 taxiFare;							 
} 
 
 main()									 
{ 
	int distance = 0, taxiFare = 0; 
 
	distance  								 
		   getDistance();		  
 
	taxiFare  								 
		   calculateTaxiFare( distance );	  
 
	printf( "%dm 이동시 택시요금은 %d원입니다 ", distance, taxiFare ); 
	 
	 0;								 
}
